# centos6 + apache + php

[Docker Hub](https://hub.docker.com/repository/docker/oppara/centos6-apache-php)

## Supported tags

* 71, latest
* 70


## build

```
$ docker build -t oppara/centos6-apache-php .
```

or

```
$ docker build -t oppara/centos6-apache-php:TAG .
```

## compose

```
$ docker-compose up
$ open http://127.0.0.1:8000
```
